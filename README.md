# RWA Team Hub

Shared workspace for the KAIST Oracle RWA team.

This repository is managed through the RWA OpenClaw instance and GitHub pull requests.

## Working Rules

- Do not push directly to `main` after the initial bootstrap.
- Create branches as `rwa/{short-slug}`.
- Open a pull request for every code or document change.
- Record the Telegram requester in commit messages or PR descriptions when work starts from Telegram.
- Keep secrets out of the repository.

## Directory Layout

- `docs/prd/` - product requirements and project briefs
- `docs/design/` - architecture and implementation designs
- `docs/meetings/` - meeting notes and decisions
- `docs/research/` - RWA research notes and source-backed memos
